🤖 Advanced Telegram Tag Bot
A production-ready Telegram tagging bot with multiple tagging modes, human-like messages, admin controls, broadcast system, stats, logging, and anti-spam protection.
Built for large groups, safe usage, and easy deployment 🚀

✨ Features
🔔 Advanced Tagging Modes
/admin or @admin → Tag only group admins with custom message
/all or @all → Tag all members in batches (7 users per message)
/tagall → One-by-one tagging with random Hinglish messages
/entag → One-by-one tagging with English messages
/hitag → One-by-one tagging with Hindi messages
/jtag → One-by-one tagging with funny + flirty Hinglish messages
✔ Messages do not repeat immediately
✔ Auto-delete tag messages
✔ Human-like delay (anti-ban safe)


🛑 Control Commands
/stop → Stop any running tag process
/resume → Resume tagging
Admin-only execution (group safety)

📢 Broadcast System (Owner Only)
Broadcast to all users who started the bot
Broadcast to all groups where bot is added
MongoDB-powered (restart safe)
